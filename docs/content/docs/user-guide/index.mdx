# User Guide

Welcome to your next-forge application! This comprehensive guide will help you understand and use all the features available in your application.

## What is next-forge?

next-forge is a production-grade application template that provides a complete foundation for building modern SaaS applications. It includes everything you need to get started quickly while maintaining high quality and scalability.

## Quick Start

Your application consists of several components:

- **Main App** (localhost:3000) - The primary user interface where you'll spend most of your time
- **Marketing Website** (localhost:3001) - Public-facing website with information about your service
- **API** (localhost:3002) - Backend services for data processing and integrations
- **Documentation** (localhost:3004) - Technical documentation and guides
- **Email Preview** (localhost:3003) - Preview and test email templates
- **Component Library** (localhost:6006) - UI component showcase

## Getting Started

1. **Sign Up/Login**: Create your account or sign in to access the application
2. **Complete Profile**: Set up your user profile and organization
3. **Explore Dashboard**: Familiarize yourself with the main interface
4. **Invite Team Members**: Add colleagues to your organization
5. **Configure Settings**: Customize your preferences and notifications

## Navigation

The application uses a sidebar navigation system with the following main sections:

- **Dashboard** - Overview and key metrics
- **Data Management** - Create, view, and manage your data
- **Team** - Manage team members and collaboration
- **Settings** - Account and application preferences
- **Billing** - Subscription and payment management
- **Help** - Documentation and support resources
